# Machine Learning labs
## 1. Project setup
### Download project:
```
git clone https://github.com/FryMondo/ML_Labs.git
```
## 2. Project initialization
### Initialize Python:
```
python -m venv .venv
```
## 3. Install required packages:
### Run next commands:
```
pip install numpy
pip install pandas
pip install matplotlib
pip install scikit-learn
```
